# 🏁 Campo Minado - Flutter

Um jogo de Campo Minado desenvolvido em Flutter.

## 📌 Tecnologias Utilizadas

- Flutter
- Dart

## 🚀 Como Executar

1. **Clone o repositório:**

   ```sh
   git clone https://github.com/erizoka/campo_minado.git
   cd campo_minado
   ```

2. **Instale as dependências:**

   ```sh
   flutter pub get
   ```

3. **Execute o app:**
   ```sh
   flutter run
   ```

## 🎮 Como Jogar

- Toque em um campo para revelá-lo.
- Evite as minas!
- Se encontrar uma mina, o jogo termina.
- Use a lógica para marcar os campos suspeitos.
